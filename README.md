# MindLab: Theory Challenge
## Overview
MindLab: Theory Challenge is an educational game designed to support mechanical engineering students in learning theoretical concepts through gamification. The game combines quizzes, card-based challenges, image recognition tasks, and puzzles to create an interactive learning experience.
The project was developed at HTW Berlin as part of the Quality Games initiative.

## Core Features

Four main mechanical engineering theory topics

Main card-based game mode combining all topics

Multiple challenge types:

Multiple-choice questions

Text input answers

Image-to-term matching

Topic-based performance results and progress tracking

Topic-specific levels for focused learning

Image description challenges

Puzzle levels with scheme assembly and direct links to related theory

## Technology

Engine: Unity

Platform: WebGL

Language: C#
